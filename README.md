# The App for Empowering the nation 
This is app is meant to explain to he user what course Empowering SA offers for the user when it come to he development of their skills.
The app also informs the user on what the course themselve will contain, when it come what the student will be taught. In that same vein of enlightning, the app will allow the user avenues for the to contact the campus directly with links to Google Map. Finaly the user also has the option to see select course and calculate how much they would cost.

## What the App does 
1. Content
   - The content is meant to inform the user of what this organization does and what service they offer. It also tells the user where to find the venues of service in order to use the business product.
2. Navigation
   - The is a mixture of navigations in the app mainly the drawer navigation and the bottom tab navigation
   - The drawer navigation shows and navigate to the four main pages i.e. Home, Calculate, Courses,  and Contacts.
   - The Drawer navigation shows the pages for the  courses and only when you switch on the toggles, and at the top there is the nav for is a Main tab which navigate back to the main page.
   - The back button returns the user back to the page from which they navigated.
3. Button Functionalities 
 - The Google Maps images
   - Images are meant to send the user to Google maps and open the map location in Google Maps.
 - The plus image buttons 
   - The plus button adding to the logged items space the course the user has selected 
 - The mini stop sign 
   - This allows the user to remove the items they want from the list in the logged items
